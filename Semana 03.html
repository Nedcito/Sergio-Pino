Semana 03

Aquí te voy a contar todo de la semana 03 sobre
Proponiendo proyectos de ODS y configurando Visual y github

Programando nuestra bitacora digital

Esta semana seguimos viendo soluciones innovadoras para diferentes ODS y sincronizamos Visual con github para empezar a programar desde nuestra PC

Clase Presencial

Comenzaba la clase y el profesor empezo a enseñarnos sobre el "href" que es algo que nos facilita mucho a hacer referencia a otra pagina, link, imagen, video lo que sea en realidad. Es decir que si nosotros colocamos "href=nombre de la carpeta/nombre de la imagen.png" o directamente el nombre de la imagen dependiendo el como lo tengamos guardado podemos agregarla a nuestra bitacora pero no solo eso si tambien colocamos "href" seguido de un link o de algun .html de nuestra bitacora dando un solo click podemos dirigirnos a ese lugar. El mayor ejemplo es el menu que tenemos en nuestra plantilla que esta llenos de "href".

Tambien como todas las semanas el profesor le da bastante importancia a lo que es idear proyectos que solucionen alguna ODS, y esta no fue la excepcion. El profesor nos dejo como actividad pensar la mayor cantidad de ideas sobre proyectos tecnologicos y no tecnologicos que podemos en el menor tiempo posible para que el luego pase a criticarlos grupo por grupo. Lamentablemente esta actividad no la pudimos ganar ya que habian algunas ideas que habiamos planteado que no estaban del todo claras pero de igual forma, alguna de las que hicimos en lo personal me gustaron mucho como por ejemplo le pusimos "Tinder de mascotas" la verdad suena muy raro pero consiste en una aplicacion tipo "Tinder" en la cual albergues, perreras, veterinarias entre otras suban fotos de las mascotas que tienen para que asi los usuarios den "match" y puedan adoptar a alguno de estos animalitos.

Clase Virtual

Empezamos la clase instalando lo que al parecer seria una extension para el Visual Studio Code que nos ayudaria a sincronizar nuestro Github con este mismo, se llama Git Bash. Una vez instalado el profesor nos explico que con ayuda del Visual Studio podriamos modificar el Github sin necesidad de estar conectados a la red unicamente copiando el link de nuestro github en el Visual Studio, y tal cual lo dijo el profesor se hizo realidad. Empezamos a ver todos los codigos de github en el Visual studio y el profesor empezo poco a poco a explicarnos como se tenian que hacer las cosas para empezar a modificar la bitacora y esta vez ya desde Visual y tambien desde nuestra propia PC

Luego teniamos que pasar a la parte en la cual teniamos que pasar todos los cambios que habiamos hecho en el Visual Studio Code al Github ya que todo lo que estabamos haciendo era unicamente visto por nosotros cuando presianabamos F5 o utilizabamos la opcion de Run. Utilizando el terminal y poniendo los codigos que el profesor nos proporciono en la clase logre sincronizar el Visual con el Github y con cada cambio que hacia me sale una notificacion en la barra de herramientas izquierda en la cual solo tengo que colocar un nombre al cambio y luego sincronizarlo. Al inicio cuesta acostumbrarte un poco y puedes llegar a creer que las cosas no estan bien configuradas pero como si fuera magia al entrar al link del Github todas las modificaciones que se han hecho luego de esperar a que carguen se suben a la nube.
