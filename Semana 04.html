Preparandonos para el proyecto final y aprendiendo CSS

Esta semana el profesor empezo a explicarnos mas sobre lo que es CSS y de esta forma poder modificar algunas cosas en nuestra bitacora asi como una dinamica en clase sobre proyectos con arduino.

Clase Presencial

En esta clase el profesor empezo a mostrarnos en vivo sobre el tema de CSS y como funcionaba nos comentaba que no podia ahondar mucho en este tema, no porque no quisiera, si no mas bien por temas de tiempo y al empezar la explicacion y modificaciones la verdad si note el porque lo decia CSS es mucho mas extenso que programar con HTML, ya que como el mismo nos lo mostro en el Visual Studio podiamos apreciar en el lado derecho una interminable lista de codigos al momento de entrar a CSS, dandonos a entender que es bastante extenso pero que a su vez se puede modificar muchas cosas como los colores, tamaños, fuentes entre otras cosas.

Pero ya que mencionamos lo de las fuentes el profesor tambien nos enseño a descargar fuentes desde Google Fonts explicandonos que era un lugar gratuito con muchas fuentes optimizadas especificamente para google, es decir que funcionaria mucho mejor desde el navegador de google. Luego de esto el profesor empezo a reemplazar en el Visual Studio las fuentes que ya tenia en la plantilla que habia creado por las nuevas fuentes que habia descargado desde google fonts y que efectivamente se reemplazaba en la misma bitacora

Luego el profesor dio un break para poder almorzar y luego del mismo el profesor dejo una actividad en la cual teniamos que pensar sobre una problematica la cual teniamos que irla resolviendo mediante lo que el profesor momentos previos nos habia explicado que es el Desing Thinking, luego de realizar la actividad, que por cosas de la vida pero la verdad merecido creo yo "jeje" mi grupo gano la actividad proponiendo como problema principal "problemas para tomar pastillas con los adultos mayores" pues nos empezamos a ver de una forma mas clara de como se tenia que empezar a trabajar este tipo de proyectos para poder llegar con una solucion efectiva.

Clase Virtual

En esta clase virtual yo tuve una urgencia de ultimo minuto y no pude asistir a la clase en vivo y en directo. Pero gracias a que las clases se graban en el moodle pude verla luego aunque no se si solo sea un error mio pero se escucho entrecortada la clase. En esta clase que fue algo corta con respecto a enseñansa el profesor nos enseña a como apoyarnos de lo que seria Chat GPT para poder programar en HTML y CSS, utilizando un promp el cual es "Hola ChatGPT, necesito que seas un desarrollador web y docente para que me ayudes a dar mis primeros pasos en programación web con HTML y CSS. Puedo iniciar a consultarte?".

De esta forma logramos que Chat GPT por decirlo de alguna manera se vuelva nuestro profesor ya que nos empieza a enseñar paso a paso como empezar con HTML y CSS logrando algo bastante similar a lo que hicimos en las primeras semanas que era el famoso "Hola mundo" que hicimos en github en la Semana 1 pero con la diferencia que esta vez todo nos lo estaba enseñando una inteligencia artificial, de manera gratuita y con el plus que nos permitia ver poco a poco lo que ibamos modificando como el tamaño del texto, un color de fondo entre otras cosas mas.

Tambien el profesor nos hizo un reforzamiento sobre como funcionaba las paginas web de hoy en dia con los lenguajes 3 lenguajes de programacion que son HTML que se encarga de la estructura y el contenido que tendra la pagina web, CSS para el diseño y JavaScript para darle vida a la pagina web con interacciones dinamicas. Tambien nos enseño a usar Netlify que es una pagina que nos ayuda a alojar la futura pagina web que haremos, de manera muy sencilla solo creando una cuenta y subiendo los archivos para luego obtener el link de nuestra pagina web.

Para finalizar el profesor mando a salas para poder ir avanzando la bitacora y tambien poder ir criticando y resolviendo dudas de todas las personas que tuvieran alguna
